Activity Logging

Tracks user activity and actions performed within the app for diagnostics and auditing purposes.

Logging is enabled via the App Setting EnableActivityLogging. When enabled, this tracks each page a user visits and the actions a user performs. This also tracks actions performed via the API.

The Activity Log shows very raw data about what a user was viewing or doing. You will need to inspect the log carefully to identify what occured. However, the data can be used for quickly identifying who did what.

The Activity Log is only accesible to users with the Administrator permission.
